# مساعد عقد SAI — بالذكاء الاصطناعي الحقيقي (مجاني وآمن)

## كيف يشتغل
- الصفحة (`index.html`) تبعث سؤال الزبون إلى `/.netlify/functions/sai-chat`.
- الفنكشن (`netlify/functions/sai-chat.js`) يبعثه لـ **Groq API** (مجاني) مع كامل تعليمات العقد كـ system prompt، ويرجع الجواب.
- **مفتاح الـ API ما يظهر أبداً في المتصفح** — مخبّى فقط في إعدادات Netlify.
- لو السيرفر أو الـ AI تعطل لأي سبب (انقطاع نت، تجاوز الحد المجاني، إلخ)، الصفحة ترجع تلقائياً للنظام الأوفلاين (مطابقة الكلمات المفتاحية) باش الزبون يوصلو جواب دايماً وما يبقاش بلا رد.

## خطوات التفعيل (5 دقائق)

1. **دير حساب Groq مجاني**: روح لـ https://console.groq.com → سجل بالإيميل → من "API Keys" أنشئ مفتاح جديد وانسخو.

2. **حمّل المشروع على Netlify**:
   - لو عندك موقع Netlify قائم: زيد فيه ملفين `netlify/functions/sai-chat.js` و `netlify.toml` (موجودين هنا)، وبدّل `index.html` بالنسخة الجديدة.
   - لو مشروع جديد: اربط المجلد بـ GitHub أو ادراغ فولدر مباشرة فـ Netlify (Deploys → Drag and drop).

3. **زيد المفتاح كمتغير بيئة (Environment Variable)**:
   - Netlify Dashboard → Site settings → Environment variables → Add a variable
   - Key: `GROQ_API_KEY`
   - Value: المفتاح اللي نسخته من Groq
   - احفظ، ثم أعد نشر الموقع (Trigger deploy) باش المتغير يتفعل.

4. **جرب**: افتح الموقع، اطرح أي سؤال بالعربي أو الإنجليزي عن العقد.

## الأمان والتكلفة
- **مجاني بالكامل** ضمن حدود Groq المجانية (سخية بزاف لحجم استخدام موقع شركة عادي).
- **آمن**: المفتاح فالسيرفر فقط، الزبون ما يقدر يوصلولو أو يسرقو.
- حددت طول الرسائل (800 حرف) وعدد الرسائل بالتاريخ (6) باش نتفادى إساءة الاستخدام أو استنزاف الحصة المجانية.
- لو حبيت تبدل الموديل مستقبلاً (مثلاً Gemini بدل Groq)، بدّل فقط دالة الـ fetch جوا `sai-chat.js` — الواجهة (index.html) ما تحتاج أي تغيير.
